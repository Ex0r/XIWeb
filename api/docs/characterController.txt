characterController Endpoint listings and details

There are several endpoints available in the characterController:

character/list 
---------------

List all of the characters associated with your account

Return values:

array[
  charid, 
  charname
]

character/<name>/view
----------------------

View the full details of the specified character

Return values:

array[
  accid, 
  charid, 
  charname,
  nation,
  pos_zone, 
  gmlevel,
  isnewplayer,
  mentor,
  campaign_allegiance,
  isstylelocked,
  array(jobs)[
    unlocked,
    genkai, 
    war, 
    mnk,
    whm, 
    blm,
    rdm,
    thf,
    pld,
    drk,
    bst,
    brd,
    rng,
    sam,
    nin,
    drg,
    smn,
    blu,
    cor,
    pup,
    dnc,
    sch,
    geo,
    run,
  ]
  array(look)[
    charid,
    face,
    race,
    size,
    head,
    body,
    hands,
    legs,  
    feet,
    main,
    sub,
    ranged

  array(profile)[
    charid,

    rank_sandoria,

    fame_windurst,

  slotid,
  equipslotid,
  containerid,
]
